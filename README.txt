NASIL KURULUR? (Chrome, Opera, Edge)

1. Bu klasörü bilgisayarınızda bir yere kaydedin (ZIP ise klasöre çıkartın).
2. Tarayıcınızı açın ve adres çubuğuna şunu yazın:
   - Chrome için: chrome://extensions
   - Opera için: opera://extensions
   - Edge için: edge://extensions
3. Sağ üst köşedeki "Geliştirici Modu" (Developer Mode) anahtarını açın.
4. "Paketlenmemiş öğe yükle" (Load Unpacked) butonuna tıklayın.
5. "FilmResumePro" klasörünü seçin.

ÖNEMLİ AYAR:
Eğer Gizli Sekme'de (Incognito) film izliyorsanız:
- Eklentiler sayfasında bu eklentinin "Ayrıntılar" butonuna tıklayın.
- "Gizli modda izin ver" seçeneğini mutlaka açın.

İyi seyirler!
